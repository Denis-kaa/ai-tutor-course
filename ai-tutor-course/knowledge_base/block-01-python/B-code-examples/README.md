# Раздел B: Примеры кода с комментариями

Цель: 20 000 слов. Минимум 100 примеров кода, каждый с построчными комментариями и разбором.

## Структура каждого примера

1. **Задача** — что нужно сделать (1-2 предложения)
2. **Код** — рабочий пример, КАЖДАЯ строка с комментарием
3. **Разбор** — что произошло и почему именно так
4. **Вариации** — как можно решить иначе
5. **Частые ошибки** — что идёт не так у новичков
