# Строки в Python

## Что такое строка

Строка (str) — это последовательность символов, заключённая в кавычки. В Python строки являются неизменяемыми (immutable): ты не можешь изменить отдельный символ в строке, но можешь создать новую строку на основе старой.

```python
text1 = "Привет, мир!"  # Двойные кавычки
text2 = 'Python'         # Одинарные кавычки
text3 = """Многострочный
текст"""                 # Тройные кавычки
```

## Аналогии строки

#### 📿 Аналогия 1: Нить с бусинами
Строка — это нить, на которую нанизаны бусины (символы). Ты можешь посмотреть на конкретную бусину по её номеру (индекс), отрезать кусок нити (срез), но не можешь заменить одну бусину на другую, не пересобрав нить заново.

#### 🚂 Аналогия 2: Вагоны поезда
Каждый символ — это вагон. У первого вагона номер 0, у второго — 1. Ты можешь сцепить два поезда (конкатенация), но не можешь вырвать вагон из середины, не разъединив состав.

#### 📖 Аналогия 3: Страница в книге
Строка — это текст на странице. Ты можешь найти слово (поиск), узнать длину текста (len), но не можешь стереть одну букву, не переписав страницу.

#### 🧬 Аналогия 4: ДНК-цепочка
Как ДНК состоит из неизменяемых нуклеотидов, так и строка состоит из фиксированных символов. Любое "изменение" — это создание новой цепочки.

#### 🏷️ Аналогия 5: Штрихкод
Строка — как штрихкод товара. Это фиксированная последовательность символов, которую можно прочитать, проверить на наличие подстроки, но нельзя частично изменить.

## Индексация и срезы

Python позволяет обращаться к символам по индексу (начиная с 0) и делать срезы.

```python
word = "Python"
print(word[0])    # 'P' (первый символ)
print(word[-1])   # 'n' (последний символ)
print(word[0:2])  # 'Py' (срез от 0 до 2, не включая 2)
print(word[::2])  # 'Pto' (каждый второй символ)
print(word[::-1]) # 'nohtyP' (переворот строки)
```

## Основные методы строк

Поскольку строки неизменяемы, методы возвращают **новую** строку.

```python
text = "  Привет, Мир!  "

# Регистр
print(text.lower())       # "  привет, мир!  "
print(text.upper())       # "  ПРИВЕТ, МИР!  "
print(text.title())       # "  Привет, Мир!  "

# Очистка
print(text.strip())       # "Привет, Мир!" (убирает пробелы по краям)

# Поиск и замена
print(text.find("Мир"))   # 9 (индекс начала подстроки, или -1)
print(text.replace("Мир", "Python")) # "  Привет, Python!  "

# Разделение и соединение
words = "яблоко,банан,вишня"
print(words.split(","))   # ['яблоко', 'банан', 'вишня']

fruits = ['яблоко', 'банан']
print(" и ".join(fruits)) # "яблоко и банан"
```

## Зачем это нужно

Без методов строк пришлось бы писать сложные циклы для каждой мелкой задачи: поиска слова, очистки пробелов или перевода в верхний регистр. Методы делают код чистым, читаемым и быстрым.

## Итого

Строка — неизменяемая последовательность символов.
Индексация начинается с 0, отрицательные индексы считают с конца.
Методы (`lower`, `strip`, `split`, `replace`) возвращают новую строку, не меняя исходную.
