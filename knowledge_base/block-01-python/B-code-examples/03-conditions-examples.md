# Примеры кода: Условные конструкции

## Пример 1: Простое условие if

**Задача:** Проверить, совершеннолетний ли человек.

```python
# Возраст человека
age = 20                    # Проверяем возраст 20 лет

# Проверяем условие: возраст больше или равен 18
if age >= 18:               # Если условие True — выполняем блок ниже
    print("Доступ разрешён")  # Этот код выполнится (20 >= 18)
    print("Добро пожаловать!")  # Этот тоже (тот же блок if)

# Этот код выполнится в любом случае (он вне блока if)
print("Проверка завершена")  # Всегда печатается
```

**Разбор:**
После `if условие:` ставится двоеточие. Весь код с отступом (4 пробела) ниже — это тело `if`, он выполняется только при истинном условии. Код без отступа выполняется всегда.

## Пример 2: if-else — два варианта

**Задача:** Выдать разный результат в зависимости от условия.

```python
# Температура на улице
temperature = -5            # Текущая температура

# Проверяем: тепло или холодно
if temperature > 0:         # Если температура выше нуля
    print("Тепло, можно без куртки")  # НЕ выполнится (-5 не > 0)
else:                       # Иначе (температура <= 0)
    print("Холодно, надень куртку!")  # Выполнится: Холодно, надень куртку!

# Всегда выполняется ровно один блок: либо if, либо else
```

**Разбор:**
`else` выполняется только если условие `if` ложно. Всегда срабатывает ровно один из двух блоков — третьего не дано. `else` не имеет условия, он "ловит" всё, что не попало в `if`.

## Пример 3: if-elif-else — несколько вариантов

**Задача:** Поставить оценку по баллам.

```python
# Баллы студента
score = 85                  # Набрал 85 баллов

# Проверяем условия СВЕРХУ ВНИЗ, останавливаемся на первом True
if score >= 90:             # 85 >= 90? Нет
    grade = "5 (отлично)"   # Пропускаем
elif score >= 75:           # 85 >= 75? Да! Останавливаемся здесь
    grade = "4 (хорошо)"    # Выполнится этот блок
elif score >= 60:           # Уже не проверяется (остановились выше)
    grade = "3 (удовл.)"    # Пропускаем
else:                       # Тоже не проверяется
    grade = "2 (неуд.)"     # Пропускаем

# Выводим результат
print(f"Оценка: {grade}")   # Оценка: 4 (хорошо)
```

**Разбор:**
Python проверяет условия сверху вниз и выполняет ПЕРВЫЙ истинный блок, после чего пропускает все остальные `elif` и `else`. Поэтому порядок важен: от большего к меньшему. Если поставить `>= 60` первым, все с 60+ баллами получат "3".

## Пример 4: Вложенные условия

**Задача:** Проверить доступ с учётом нескольких факторов.

```python
# Исходные данные
age = 25                    # Возраст
has_ticket = True           # Есть билет
is_vip = False              # Не VIP

# Внешнее условие: есть ли билет?
if has_ticket:              # Билет есть — заходим внутрь
    # Вложенное условие: VIP или обычный?
    if is_vip:              # VIP?
        print("Проходите в VIP-зону")  # Не выполнится
    else:                   # Не VIP
        print("Проходите в обычный зал")  # Выполнится
else:                       # Билета нет
    print("Купите билет в кассе")  # Не выполнится (билет есть)
```

**Разбор:**
Условия можно вкладывать друг в друга. Но глубокая вложенность (3+ уровня) ухудшает читаемость. Часто лучше использовать логические операторы `and`/`or` вместо вложенности.

**Плоский вариант (лучше):**
```python
# Тот же результат без вложенности
if has_ticket and is_vip:
    print("Проходите в VIP-зону")
elif has_ticket:
    print("Проходите в обычный зал")
else:
    print("Купите билет в кассе")
```

## Пример 5: Тернарный оператор

**Задача:** Присвоить значение в зависимости от условия в одну строку.

```python
# Возраст
age = 20                    # Проверяем возраст

# Обычный if-else (3 строки)
if age >= 18:
    status = "взрослый"
else:
    status = "ребёнок"

# Тернарный оператор (1 строка) — тот же результат
status = "взрослый" if age >= 18 else "ребёнок"
print(status)               # взрослый

# Ещё примеры тернарного оператора
number = 7
parity = "чётное" if number % 2 == 0 else "нечётное"
print(parity)               # нечётное

# Тернарник можно вкладывать (но не злоупотребляй!)
score = 85
grade = "5" if score >= 90 else "4" if score >= 75 else "3"
print(grade)                # 4
```

**Разбор:**
Синтаксис: `значение_если_да if условие else значение_если_нет`. Удобно для простых присваиваний. Для сложной логики лучше обычный if-else — он читаемее.

## Пример 6: Проверка нескольких условий с and/or

**Задача:** Проверить, подходит ли кандидат на вакансию.

```python
# Данные кандидата
age = 30                    # Возраст
experience = 5              # Опыт работы в годах
education = "высшее"        # Образование
knows_python = True         # Знает Python

# Условия вакансии:
# Возраст 25-40, опыт >= 3, высшее образование, знает Python
is_suitable = (
    25 <= age <= 40         # Возраст в диапазоне
    and experience >= 3     # И опыт достаточный
    and education == "высшее"  # И образование подходит
    and knows_python        # И знает Python
)

# Выводим результат
if is_suitable:             # Если все условия True
    print("Кандидат подходит! Приглашаем на собеседование.")
else:                       # Если хотя бы одно False
    print("Кандидат не соответствует требованиям.")

# Кандидат подходит! Приглашаем на собеседование.
```

**Разбор:**
Скобки группируют условия для читаемости. `and` требует истинности всех условий. Если хотя бы одно ложно — всё выражение ложно. Многострочная запись в скобках делает сложное условие читаемым.

## Частые ошибки новичков

| Ошибка | Пример | Решение |
|--------|--------|---------|
| Забыл двоеточие | `if x > 5` | `if x > 5:` |
| Нет отступа | `if x>5:` + `print()` без отступа | 4 пробела перед телом |
| `=` вместо `==` | `if x = 5:` | `if x == 5:` |
| Неправильный порядок elif | `>=60` перед `>=90` | От большего к меньшему |
| Вложенность 4+ уровня | if в if в if в if | Используй and/or |
