# Примеры кода: Операторы

## Пример 1: Арифметические операторы

**Задача:** Выполнить все базовые математические операции.

```python
# Исходные числа
a = 17                      # Первое число
b = 5                       # Второе число

# Сложение
sum_result = a + b          # 17 + 5 = 22
print("Сумма:", sum_result)  # Сумма: 22

# Вычитание
diff = a - b                # 17 - 5 = 12
print("Разность:", diff)     # Разность: 12

# Умножение
product = a * b             # 17 * 5 = 85
print("Произведение:", product)  # Произведение: 85

# Обычное деление (всегда возвращает float!)
quotient = a / b            # 17 / 5 = 3.4
print("Деление:", quotient)  # Деление: 3.4

# Целочисленное деление (отбрасывает дробную часть)
floor_div = a // b          # 17 // 5 = 3 (не 3.4!)
print("Целочисленное деление:", floor_div)  # Целочисленное деление: 3

# Остаток от деления (модуль)
remainder = a % b           # 17 % 5 = 2 (17 = 5*3 + 2)
print("Остаток:", remainder)  # Остаток: 2

# Возведение в степень
power = a ** b              # 17 ** 5 = 1419857
print("Степень:", power)     # Степень: 1419857
```

**Разбор:**
Главная ловушка — оператор `/` всегда возвращает `float`, даже если делится нацело: `10 / 2 = 5.0`. Если нужно целое число — используй `//`. Остаток `%` незаменим для проверки чётности и циклических операций.

## Пример 2: Проверка чётности через остаток

**Задача:** Определить, чётное число или нечётное.

```python
# Число для проверки
number = 7                  # Проверяем число 7

# Вычисляем остаток от деления на 2
remainder = number % 2      # 7 % 2 = 1 (остаток)

# Если остаток 0 — число чётное, иначе нечётное
if remainder == 0:          # Остаток равен нулю?
    print(f"{number} — чётное")     # Этот блок НЕ выполнится
else:                       # Остаток не равен нулю
    print(f"{number} — нечётное")   # 7 — нечётное (выполнится)

# Короткая запись через тернарный оператор
result = "чётное" if number % 2 == 0 else "нечётное"
print(result)               # нечётное
```

**Разбор:**
`number % 2` даёт 0 для чётных и 1 для нечётных чисел. Это самый частый способ проверки чётности. Тернарный оператор `X if условие else Y` позволяет записать if-else в одну строку.

## Пример 3: Операторы сравнения

**Задача:** Сравнить значения и получить логический результат.

```python
# Исходные значения
x = 10                      # Первое число
y = 5                       # Второе число

# Равно (==)
print(x == y)               # False (10 не равно 5)
print(x == 10)              # True (10 равно 10)

# Не равно (!=)
print(x != y)               # True (10 не равно 5)

# Больше (>)
print(x > y)                # True (10 больше 5)

# Меньше (<)
print(x < y)                # False (10 не меньше 5)

# Больше или равно (>=)
print(x >= 10)              # True (10 больше ИЛИ равно 10)
print(x >= 11)              # False

# Меньше или равно (<=)
print(y <= 5)               # True (5 меньше ИЛИ равно 5)

# Цепочка сравнений (фишка Python!)
age = 25
# Вместо: age >= 18 and age <= 65
is_working_age = 18 <= age <= 65    # Читается как в математике
print(is_working_age)       # True
```

**Разбор:**
Операторы сравнения возвращают `True` или `False`. Python поддерживает цепочки сравнений: `18 <= age <= 65` эквивалентно `age >= 18 and age <= 65`, но читается естественнее.

## Пример 4: Логические операторы

**Задача:** Комбинировать несколько условий.

```python
# Исходные данные
age = 25                    # Возраст
has_license = True          # Есть права
has_car = False             # Нет машины
is_sober = True             # Трезвый

# AND — оба условия должны быть True
can_drive = age >= 18 and has_license and is_sober
print("Может водить:", can_drive)  # Может водить: True

# OR — хотя бы одно условие True
has_transport = has_car or has_license
print("Есть транспорт:", has_transport)  # Есть транспорт: True (права есть)

# NOT — инверсия (переворачивает значение)
is_minor = not (age >= 18)
print("Несовершеннолетний:", is_minor)  # Несовершеннолетний: False

# Комбинация: можно ехать, если (взрослый И трезвый И есть права) ИЛИ (есть такси)
has_taxi = False
can_go = (age >= 18 and is_sober and has_license) or has_taxi
print("Можно ехать:", can_go)  # Можно ехать: True
```

**Разбор:**
`and` требует истинности ВСЕХ условий. `or` требует истинности ХОТЯ БЫ ОДНОГО. `not` переворачивает значение. Скобки меняют приоритет: без них `not` выполняется первым, потом `and`, потом `or`.

## Пример 5: Приоритет операторов

**Задача:** Показать, как скобки меняют результат.

```python
# Без скобок: умножение выполняется раньше сложения
result1 = 2 + 3 * 4         # 2 + (3*4) = 2 + 12 = 14
print(result1)              # 14

# Со скобками: сложение выполняется первым
result2 = (2 + 3) * 4       # (2+3) * 4 = 5 * 4 = 20
print(result2)              # 20

# Логические операторы: not > and > or
# Без скобок
result3 = True or False and False   # True or (False and False) = True or False = True
print(result3)              # True

# Со скобками — меняем порядок
result4 = (True or False) and False  # (True) and False = False
print(result4)              # False

# Сложное выражение с арифметикой и сравнением
x = 10
# Степень > умножение > сложение > сравнение
result5 = x ** 2 > 50 + 10  # (10**2) > (50+10) → 100 > 60 → True
print(result5)              # True
```

**Разбор:**
Порядок приоритетов: `**` → `*`, `/`, `//`, `%` → `+`, `-` → сравнения → `not` → `and` → `or`. Когда сомневаешься — ставь скобки. Они делают код читаемым и исключают ошибки.

## Частые ошибки новичков

| Ошибка | Пример | Решение |
|--------|--------|---------|
| `=` вместо `==` | `if x = 5:` | Используй `==` для сравнения |
| Деление даёт float | `10 / 2` → `5.0` | Используй `//` для целого |
| Забыл скобки | `2 + 3 * 4` ≠ `(2+3)*4` | Ставь скобки явно |
| `and`/`or` с числами | `5 and 3` → `3` | Помни: возвращают операнд, а не bool |
