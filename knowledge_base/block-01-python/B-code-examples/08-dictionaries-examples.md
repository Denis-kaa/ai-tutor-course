# Примеры кода: Словари

## Пример 1: Создание и доступ по ключу

**Задача:** Создать словарь и доставать значения по ключам.

```python
# Создаём словарь как набор пар "ключ: значение"
person = {
    "name": "Алексей",       # Ключ "name" -> значение "Алексей"
    "age": 30,               # Ключ "age" -> значение 30
    "city": "Москва"         # Ключ "city" -> значение "Москва"
}

# Доступ по ключу через квадратные скобки
print(person["name"])       # Алексей
print(person["age"])        # 30

# Изменение значения по существующему ключу
person["age"] = 31          # Теперь age = 31

# Добавление НОВОГО ключа (просто присваиваем)
person["email"] = "alex@mail.ru"   # Новый ключ "email"

print(person)
# {'name': 'Алексей', 'age': 31, 'city': 'Москва', 'email': 'alex@mail.ru'}
```

**Разбор:**
Словарь — коллекция пар ключ-значение. Ключи уникальны и неизменяемы (строки, числа, кортежи). Доступ `dict[key]` быстрый (O(1)). Присваивание по новому ключу добавляет пару, по существующему — меняет значение.

## Пример 2: Безопасный доступ через get()

**Задача:** Не падать с ошибкой при обращении к несуществующему ключу.

```python
user = {"name": "Мария", "age": 25}   # Словарь без ключа "email"

# ❌ Опасно: KeyError, если ключа нет
# print(user["email"])        # KeyError: 'email'

# ✅ Безопасно: get() вернёт None, если ключа нет
print(user.get("email"))    # None (без ошибки!)

# get() со значением по умолчанию
print(user.get("email", "не указан"))   # не указан
print(user.get("name", "не указан"))    # Мария (ключ есть — вернул реальное значение)

# Практический пример: подсчёт слов
text = "кот собака кот птица собака кот"
words = text.split()        # ['кот', 'собака', 'кот', 'птица', 'собака', 'кот']

counter = {}                # Пустой словарь-счётчик
for word in words:          # Проходим по каждому слову
    counter[word] = counter.get(word, 0) + 1   # get(word, 0) = текущий счёт или 0
    # Если слова ещё нет — get вернёт 0, прибавим 1 -> создастся ключ со значением 1

print(counter)              # {'кот': 3, 'собака': 2, 'птица': 1}
```

**Разбор:**
`get(key)` возвращает `None` вместо ошибки. `get(key, default)` возвращает default, если ключа нет. Это идиома для подсчёта и безопасного доступа. Альтернатива — `collections.defaultdict`, но `get` проще для старта.

## Пример 3: Удаление и проверка наличия

**Задача:** Удалять ключи и проверять, есть ли ключ в словаре.

```python
scores = {"Аня": 90, "Боря": 75, "Вера": 88}   # Оценки студентов

# Проверка наличия ключа через in
print("Аня" in scores)      # True
print("Гоша" in scores)     # False

# pop — удалить ключ И вернуть значение
vera_score = scores.pop("Вера")   # vera_score = 88, ключ удалён
print(vera_score)           # 88
print(scores)               # {'Аня': 90, 'Боря': 75}

# pop с default — не упадёт, если ключа нет
missing = scores.pop("Гоша", 0)   # Ключа нет -> вернёт 0, без ошибки
print(missing)              # 0

# del — удалить ключ (ошибка, если нет)
del scores["Боря"]          # Удаляем Борю
print(scores)               # {'Аня': 90}

# clear — очистить весь словарь
scores.clear()              # scores = {}
print(scores)               # {}
```

**Разбор:**
`in` проверяет наличие КЛЮЧА (не значения!). `pop` удаляет и возвращает значение — удобно, когда нужно и достать, и убрать. `pop` с default безопасен. `del` выбрасывает KeyError при отсутствии ключа. `clear` опустошает словарь на месте.

## Пример 4: Перебор словаря

**Задача:** Пройтись по ключам, значениям и парам.

```python
prices = {"молоко": 80, "хлеб": 45, "яйца": 120}   # Цены товаров

# Перебор только КЛЮЧЕЙ (по умолчанию)
for key in prices:          # То же, что for key in prices.keys()
    print(key)              # молоко / хлеб / яйца

print("---")

# Перебор только ЗНАЧЕНИЙ
for price in prices.values():
    print(price)            # 80 / 45 / 120

print("---")

# Перебор ПАР (ключ, значение) через items() — самый частый случай
for product, price in prices.items():   # Распаковка пары в две переменные
    print(f"{product} стоит {price} руб.")
# молоко стоит 80 руб. / хлеб стоит 45 руб. / яйца стоит 120 руб.

print("---")

# Перебор с индексом через enumerate + items
for i, (product, price) in enumerate(prices.items(), start=1):
    print(f"{i}. {product} — {price} руб.")
# 1. молоко — 80 руб. / 2. хлеб — 45 руб. / 3. яйца — 120 руб.
```

**Разбор:**
`keys()`, `values()`, `items()` возвращают представления словаря. `items()` даёт пары для распаковки `key, value`. Комбинация `enumerate(...items())` даёт индекс + ключ + значение — удобно для нумерованных списков.

## Пример 5: Dict comprehension и слияние

**Задача:** Создать словарь в одну строку и объединить два словаря.

```python
# Dict comprehension — квадрат чисел как словарь
squares = {x: x ** 2 for x in range(1, 6)}   # x от 1 до 5
print(squares)            # {1: 1, 2: 4, 3: 9, 4: 16, 5: 25}

# С условием: только чётные ключи
even = {x: x ** 2 for x in range(1, 11) if x % 2 == 0}
print(even)               # {2: 4, 4: 16, 6: 36, 8: 64, 10: 100}

# Инверсия словаря: поменять ключи и значения местами
original = {"a": 1, "b": 2, "c": 3}
inverted = {v: k for k, v in original.items()}   # v становится ключом
print(inverted)           # {1: 'a', 2: 'b', 3: 'c'}

print("---")

# Слияние двух словарей
defaults = {"color": "white", "size": "M", "price": 100}
user_prefs = {"color": "black", "size": "L"}     # Пользователь переопределил 2 поля

# Оператор | (Python 3.9+) — правый словарь перезаписывает левый
merged = defaults | user_prefs
print(merged)             # {'color': 'black', 'size': 'L', 'price': 100}

# update() — меняет словарь НА МЕСТЕ
defaults.update(user_prefs)
print(defaults)           # {'color': 'black', 'size': 'L', 'price': 100}
```

**Разбор:**
Dict comprehension `{key: value for ...}` создаёт словарь в одну строку. Инверсия `{v: k for k, v in d.items()}` — частый приём. Оператор `|` сливает словари (правый побеждает при конфликте ключей). `update()` делает то же на месте.

## Пример 6: Вложенные словари

**Задача:** Хранить структуру с несколькими уровнями.

```python
# База данных пользователей — словарь словарей
users = {
    "user1": {"name": "Аня", "age": 25, "role": "admin"},
    "user2": {"name": "Боря", "age": 30, "role": "user"},
    "user3": {"name": "Вера", "age": 28, "role": "user"}
}

# Доступ к вложенному значению — цепочка ключей
print(users["user1"]["name"])     # Аня
print(users["user2"]["role"])     # user

# Безопасный доступ к вложенному через get (защита от KeyError)
print(users.get("user99", {}).get("name", "не найден"))   # не найден

print("---")

# Перебор вложенного словаря
for user_id, data in users.items():       # data — вложенный словарь
    print(f"{user_id}: {data['name']}, {data['age']} лет, роль {data['role']}")
# user1: Аня, 25 лет, роль admin / ...

print("---")

# Поиск пользователей по роли
admins = [data["name"] for data in users.values() if data["role"] == "admin"]
print("Админы:", admins)    # Админы: ['Аня']
```

**Разбор:**
Вложенные словари моделируют реальные структуры (JSON из API, конфигурации). Доступ — цепочкой `d["a"]["b"]`. Для безопасности на каждом уровне используй `get(key, {})`, чтобы отсутствие ключа не ломало цепочку. `values()` даёт все вложенные словари для фильтрации.

## Частые ошибки новичков

| Ошибка | Пример | Решение |
|--------|--------|---------|
| KeyError | `d["missing"]` | Используй `d.get("missing")` |
| Список как ключ | `{[1,2]: "x"}` → TypeError | Ключи неизменяемы: кортеж, строка, число |
| Изменение при переборе | `for k in d: del d[k]` | Перебирай `list(d.keys())` |
| Путаница in | `"val" in d` ищет КЛЮЧ | Для значений: `"val" in d.values()` |
