# Переменные и типы данных в Python

## Что такое переменная

Переменная — это именованная ячейка памяти, которая хранит значение. Когда ты создаёшь переменную, Python выделяет место в оперативной памяти и связывает его с именем.

Простыми словами: переменная — это контейнер для данных. Ты даёшь контейнеру имя, кладёшь внутрь значение, и потом можешь использовать это значение по имени.

```python
age = 25
name = "Алексей"
is_student = True
```

## Аналогии из жизни

### 📦 Аналогия 1: Коробка с наклейкой

Представь, что ты переезжаешь и упаковываешь вещи в коробки. На каждую коробку ты клеишь наклейку с названием: "Книги", "Одежда", "Кухня".

Переменная — это коробка. Имя переменной — наклейка. Значение — то, что внутри.

```python
books = ["Война и мир", "Мастер и Маргарита"]  # Коробка "Книги"
clothes = ["Рубашка", "Джинсы"]  # Коробка "Одежда"
```

Когда тебе нужны книги, ты не перебираешь все коробки — ты берёшь коробку с наклейкой "Книги". Так же и в коде: `print(books)` — и ты получаешь содержимое переменной `books`.

**Что это даёт:**
- Не нужно запоминать, где лежат данные (в какой ячейке памяти)
- Достаточно знать имя переменной
- Можно менять содержимое, не меняя имя

### 🅿️ Аналогия 2: Парковочное место

Переменная — это парковочное место с номером. На место (ячейку памяти) ставится машина (значение), а номер места — это имя переменной.

```python
car_1 = "Toyota Camry"  # Место №1, машина — Toyota
car_2 = "Honda Civic"   # Место №2, машина — Honda
```

Чтобы найти машину, ты зовёшь её по номеру места: `print(car_1)` — и видишь "Toyota Camry".

**Важный момент:** На одно парковочное место можно поставить только одну машину. Если ты напишешь:

```python
car_1 = "Toyota"
car_1 = "Honda"  # Старая машина уехала, новая встала
```

То в `car_1` будет только "Honda". Старое значение перезаписалось.

### 📱 Аналогия 3: Контакт в телефоне

Переменная — как сохранённый контакт в телефоне. Имя контакта (переменной) — "мама", а внутри — номер телефона (значение).

```python
mom_phone = "+79991234567"
dad_phone = "+79997654321"
```

Ты не запоминаешь номер — ты просто вызываешь "маму". Так же и в коде: ты работаешь с именем переменной, а Python сам знает, где в памяти лежит значение.

**Преимущество:** Если номер мамы изменится, ты обновляешь контакт в одном месте, и все, кто звонит "маме", попадают на новый номер. В коде так же: изменил значение переменной — и весь код, который её использует, работает с новым значением.

### 💼 Аналогия 4: Папка на рабочем столе

Переменная — это папка на рабочем столе компьютера. Ты создаёшь папку "Отчёт за май", кладёшь туда файл. Когда нужен отчёт — открываешь папку по имени.

```python
report_may = "Данные за май.xlsx"
report_june = "Данные за июнь.xlsx"
```

**Что это даёт:**
- Порядок и структура
- Быстрый доступ по имени
- Можно менять содержимое папки, не меняя её название

### 🏷️ Аналогия 5: Бирка на багаже в аэропорту

Когда сдаёшь багаж в аэропорту, на чемодан вешают бирку с номером. Этот номер — как имя переменной. Чемодан (значение) может быть любым, но ты отслеживаешь его по номеру (имени переменной).

```python
luggage_1 = "Чемодан 20кг"
luggage_2 = "Рюкзак 5кг"
```

Когда получаешь багаж, ты называешь номер бирки — и получаешь свой чемодан. В коде: `print(luggage_1)` — и получаешь "Чемодан 20кг".

## Типы данных

В Python переменная может хранить данные разных типов. Основные типы:

### 1. Числа (int, float)

**int** (integer) — целые числа:
```python
age = 25
temperature = -10
count = 1000000
```

**float** — дробные числа (с плавающей точкой):
```python
price = 19.99
pi = 3.14159
temperature = -5.5
```

**Аналогия:** int — это целые яблоки (1, 2, 3). float — это яблоки, разрезанные на дольки (1.5, 2.7).

### 2. Строки (str)

Строка — это текст, последовательность символов:
```python
name = "Алексей"
message = "Привет, мир!"
empty = ""  # Пустая строка
```

**Аналогия:** Строка — это бусы. Каждая бусина — один символ. Ты можешь посчитать количество бусин (длина строки), взять первые 5 бусин (срез), перевернуть бусы.

### 3. Логический тип (bool)

Может принимать только два значения: `True` (истина) или `False` (ложь):
```python
is_student = True
has_license = False
is_adult = age >= 18
```

**Аналогия:** Логический тип — это выключатель. Он может быть либо включён (True), либо выключен (False). Третьего не дано.

### 4. NoneType

Специальное значение `None` означает "ничего" или "пустоту":
```python
result = None  # Пока результата нет
middle_name = None  # Отчество не указано
```

**Аналогия:** None — это пустая коробка. Коробка есть (переменная создана), но внутри ничего нет.

## Как Python понимает тип

Python — язык с динамической типизацией. Это значит, что тебе не нужно явно указывать тип переменной. Python сам определяет тип по значению:

```python
x = 10        # Python видит число без точки → int
y = 10.5      # Python видит число с точкой → float
z = "Привет"  # Python видит кавычки → str
w = True      # Python видит True/False → bool
```

**Проверить тип можно функцией `type()`:**
```python
print(type(x))  # <class 'int'>
print(type(y))  # <class 'float'>
print(type(z))  # <class 'str'>
```

## Правила именования переменных

1. **Имя может содержать буквы, цифры и символ подчёркивания `_`**
2. **Не может начинаться с цифры**
3. **Не может содержать пробелы и специальные символы** (!@#$%^&* и т.д.)
4. **Не может совпадать с зарезервированными словами Python** (if, for, while, class, def, return, import, from, True, False, None и др.)
5. **Рекомендуется использовать snake_case** (слова разделяются подчёркиванием): `my_age`, `total_price`, `is_valid`

**Примеры:**
```python
# ✅ Правильно:
age = 25
user_name = "Алексей"
total_price = 199.99
is_valid = True
_data = "скрытое"  # Начинается с подчёркивания — допустимо

# ❌ Неправильно:
# 2age = 25  # Начинается с цифры
# user-name = "Алексей"  # Содержит дефис
# total price = 199.99  # Содержит пробел
# class = "Python"  # Зарезервированное слово
```

## Зачем это нужно

Без переменных мы бы не могли:
- Сохранять результаты вычислений
- Передавать данные между частями программы
- Работать с пользовательским вводом
- Создавать сложные структуры данных

**Пример без переменных (плохо):**
```python
print(10 + 20)  # Результат выводится, но не сохраняется
print(10 + 20)  # Приходится считать заново
print(10 + 20)  # И ещё раз
```

**Пример с переменными (хорошо):**
```python
result = 10 + 20
print(result)  # Используем сохранённый результат
print(result * 2)  # Можем делать вычисления над ним
print(result + 5)  # И ещё раз
```

## Что было бы, если бы переменных не существовало

Представь, что ты пишешь программу для расчёта зарплаты. Без переменных тебе пришлось бы каждый раз вводить одни и те же числа:

```python
# Без переменных
print(40 * 500)  # 40 часов * 500 рублей/час
print(40 * 500 + 1000)  # + премия
print((40 * 500 + 1000) * 0.87)  # - налог 13%
```

Если ставка изменится, тебе придётся искать все места, где написано `500`, и менять. А если таких мест 100?

**С переменными:**
```python
hours = 40
rate = 500
bonus = 1000
tax_rate = 0.13

salary = hours * rate
total = salary + bonus
net_salary = total * (1 - tax_rate)

print(net_salary)
```

Если ставка изменится — меняешь только в одном месте: `rate = 600`. Весь остальной код работает как прежде.

## Связь с другими концепциями

Переменные тесно связаны с:
- **Операторами** (+, -, *, /) — для вычислений
- **Условиями** (if/else) — для проверки значений
- **Циклами** (for, while) — для обработки коллекций
- **Функциями** — для передачи данных как аргументов

Без понимания переменных невозможно двигаться дальше. Это фундамент всего программирования.

## Итого

Переменная — это именованный контейнер для данных. Она позволяет:
- Сохранять значения для повторного использования
- Давать данным понятные имена
- Менять значения, не меняя логику программы
- Работать с разными типами данных (числа, текст, логические значения)

Правила именования: буквы, цифры, подчёркивание, не начинать с цифры, использовать snake_case.

Типы данных определяются автоматически: int, float, str, bool, None.
