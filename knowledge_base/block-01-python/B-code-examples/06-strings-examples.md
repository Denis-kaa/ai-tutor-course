# Примеры кода: Строки

## Пример 1: Индексация и срезы

**Задача:** Доставать символы и куски строки по индексам.

```python
word = "Python"            # Строка из 6 символов (индексы 0..5)

# Доступ к одному символу по индексу (счёт с 0!)
print(word[0])              # P (первый символ)
print(word[5])              # n (последний символ, индекс = длина-1)
print(word[-1])             # n (последний через отрицательный индекс)
print(word[-2])             # o (предпоследний)

# Срезы [start:stop:step] — stop НЕ включается
print(word[0:3])            # Pyt (с 0 по 2)
print(word[2:])             # thon (с 2 до конца)
print(word[:3])             # Pyt (с начала до 2)
print(word[::2])            # Pto (каждый второй символ)
print(word[::-1])           # nohtyP (переворот строки!)
```

**Разбор:**
Индексация начинается с 0. Отрицательные индексы считают с конца (-1 = последний). Срез `[start:stop]` берёт символы от start до stop-1. Шаг `step` пропускает символы; `[::-1]` — классический трюк для переворота строки.

## Пример 2: Методы изменения регистра и очистки

**Задача:** Привести строку к нужному виду.

```python
text = "  Привет, Мир!  "   # Строка с пробелами по краям

# Изменение регистра
print(text.lower())         # "  привет, мир!  " (всё в нижний регистр)
print(text.upper())         # "  ПРИВЕТ, МИР!  " (всё в верхний)
print(text.title())         # "  Привет, Мир!  " (Каждое Слово С Заглавной)
print(text.capitalize())    # "  привет, мир!  " (только первое слово)
print(text.swapcase())      # "  пРИВЕТ, мИР!  " (инверсия регистра)

# Очистка пробелов
print(text.strip())         # "Привет, Мир!" (убрали пробелы с ОБЕИХ сторон)
print(text.lstrip())        # "Привет, Мир!  " (только СЛЕВА)
print(text.rstrip())        # "  Привет, Мир!" (только СПРАВА)

# Исходная строка НЕ изменилась (строки неизменяемы!)
print(text)                 # "  Привет, Мир!  " (как была)
```

**Разбор:**
Все методы строк возвращают НОВУЮ строку, не меняя исходную (строки immutable). `strip()` убирает пробелы и переносы строк по краям — незаменимо при обработке ввода пользователя. `lower()` часто используют для сравнения без учёта регистра.

## Пример 3: split и join

**Задача:** Разбить строку на части и собрать обратно.

```python
# split — разбивает строку в список по разделителю
fruits_str = "яблоко,банан,вишня,дыня"   # Строка через запятую
fruits_list = fruits_str.split(",")       # Разбиваем по запятой
print(fruits_list)          # ['яблоко', 'банан', 'вишня', 'дыня']

# split без аргументов — по пробелам
sentence = "я люблю python"
words = sentence.split()    # Разбиваем по пробелам
print(words)                # ['я', 'люблю', 'python']

print("---")

# join — собирает список в строку через разделитель
fruits = ["яблоко", "банан", "вишня"]
result = " и ".join(fruits) # Склеиваем через " и "
print(result)               # яблоко и банан и вишня

# join с переносом строки
lines = "\n".join(["строка 1", "строка 2", "строка 3"])
print(lines)
# строка 1
# строка 2
# строка 3
```

**Разбор:**
`split(разделитель)` превращает строку в список. `join(список)` — обратная операция, но вызывается НА РАЗДЕЛИТЕЛЕ: `"разделитель".join(список)`. Это частая путаница у новичков — join это метод строки-разделителя, а не списка.

## Пример 4: Поиск подстроки (find и in)

**Задача:** Проверить, есть ли подстрока, и найти её позицию.

```python
text = "Я люблю программировать на Python"   # Исходная строка

# Оператор in — проверяет наличие (возвращает True/False)
print("Python" in text)     # True (подстрока есть)
print("Java" in text)       # False (подстроки нет)
print("python" in text)     # False (регистр важен!)

# Поиск без учёта регистра — приводим обе строки к lower
print("python" in text.lower())  # True

print("---")

# find — возвращает индекс первого вхождения (или -1)
print(text.find("Python"))  # 27 (индекс, где начинается "Python")
print(text.find("Java"))    # -1 (не найдено)
print(text.find("лю"))      # 2 (первое вхождение "лю")

# count — сколько раз встречается подстрока
print("о".count("о"))       # Считаем букву "о"
print(text.count("на"))     # 1
```

**Разбор:**
`in` — самый питоничный способ проверки наличия подстроки. `find` возвращает позицию или -1 (не выбрасывает ошибку, в отличие от `index`). Поиск чувствителен к регистру — для нечувствительного сравнивай через `.lower()`.

## Пример 5: Три способа форматирования строк

**Задача:** Подставить переменные в строку тремя способами.

```python
name = "Алексей"            # Имя
age = 30                    # Возраст
price = 1234.5678           # Цена

# Способ 1: f-строки (РЕКОМЕНДУЕТСЯ) — переменные в {}
print(f"Меня зовут {name}, мне {age} лет")   # Меня зовут Алексей, мне 30 лет
print(f"Цена: {price:.2f} руб.")              # Цена: 1234.57 руб. (2 знака)
print(f"Сумма: {100 + 200}")                  # Сумма: 300 (выражение внутри!)

# Способ 2: метод format() — плейсхолдеры {}
print("Меня зовут {}, мне {} лет".format(name, age))
print("Имя: {0}, Возраст: {1}".format(name, age))   # По позиции
print("{n} — {a} лет".format(n=name, a=age))        # По имени

# Способ 3: оператор % (устаревший, но встречается в старом коде)
print("Меня зовут %s, мне %d лет" % (name, age))  # %s=строка, %d=число
print("Цена: %.2f" % price)                        # %.2f = float с 2 знаками
```

**Разбор:**
f-строки (Python 3.6+) — самые читаемые и быстрые, позволяют писать выражения прямо в `{}`. `format()` — гибкий, хорош для шаблонов. `%` — устаревший синтаксис из C, встречается в старом коде, в новом писать не стоит.

## Пример 6: Проверки содержимого строки

**Задача:** Проверить, из чего состоит строка.

```python
text = "Hello123"           # Тестовая строка

# Проверки типа содержимого
print(text.isdigit())       # False (есть буквы)
print("12345".isdigit())    # True (только цифры)
print(text.isalpha())       # False (есть цифры)
print("Hello".isalpha())    # True (только буквы)
print(text.isalnum())       # True (буквы И цифры, без пробелов)
print("   ".isspace())      # True (только пробелы)

print("---")

# Проверки начала и конца
email = "user@gmail.com"
print(email.startswith("user"))     # True (начинается с "user")
print(email.endswith(".com"))       # True (заканчивается на ".com")
print(email.endswith(".ru"))        # False

# Замена подстроки
greeting = "Привет, мир!"
new_greeting = greeting.replace("мир", "Python")  # Заменяем "мир" на "Python"
print(new_greeting)         # Привет, Python!
print(greeting)             # Привет, мир! (исходная не изменилась!)
```

**Разбор:**
Методы `is...` возвращают `True`/`False` и удобны для валидации ввода (проверить, что пользователь ввёл число). `startswith`/`endswith` проверяют префикс/суффикс. `replace` возвращает новую строку — исходная остаётся прежней.

## Частые ошибки новичков

| Ошибка | Пример | Решение |
|--------|--------|---------|
| Изменение символа | `s[0] = "X"` → TypeError | Создай новую строку |
| Индекс вне диапазона | `"abc"[5]` → IndexError | Проверяй длину через `len()` |
| join вызван не так | `list.join(",")` | `",".join(list)` |
| Регистр при поиске | `"Python" in "python"` → False | Сравнивай через `.lower()` |
