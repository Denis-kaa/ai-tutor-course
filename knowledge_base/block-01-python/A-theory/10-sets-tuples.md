# Множества и кортежи в Python

## Что такое кортеж (tuple)

Кортеж — это упорядоченная, но **неизменяемая** коллекция элементов. В отличие от списка, ты не можешь добавить, удалить или заменить элемент в кортеже после его создания.

```python
coordinates = (55.75, 37.61)  # Широта и долгота Москвы
rgb_color = (255, 0, 0)       # Красный цвет
single_element = (42,)        # Кортеж из одного элемента (запятая обязательна!)
```

### Аналогии кортежа

#### 📦 Аналогия 1: Запечатанная посылка
Кортеж — как запечатанная коробка с вещами. Ты можешь посмотреть, что внутри (прочитать элементы), но не можешь положить туда что-то новое или вытащить старое, не вскрыв коробку (не создав новую).

#### 📅 Аналогия 2: Дата в календаре
Дата — это кортеж: (год, месяц, день). 24 июля 2026 года — это фиксированный факт. Ты не можешь "изменить" 24 июля на 25, не создав новую дату.

```python
date = (2026, 7, 24)
print(date[0])  # 2026
# date[0] = 2027  # TypeError! Кортежи неизменяемы
```

#### 🎫 Аналогия 3: Билет на поезд
Билет содержит фиксированные данные: (вагон, место, дата). После печати билета ты не можешь поменять вагон, просто изменив цифру на бумаге. Нужно оформлять новый билет.

## Что такое множество (set)

Множество — это неупорядоченная коллекция **уникальных** элементов. В множестве не может быть дубликатов, и элементы не имеют индексов.

```python
fruits = {"яблоко", "банан", "яблоко", "вишня"}
print(fruits)  # {'яблоко', 'банан', 'вишня'} — дубликат удален автоматически
```

### Аналогии множества

#### 🛒 Аналогия 1: Корзина с уникальными товарами
Множество — как корзина в магазине, куда ты кладешь товары. Если ты случайно положил два одинаковых яблока, на кассе их всё равно пробьют как один товар. Дубликаты не имеют смысла.

#### 👥 Аналогия 2: Список гостей на вечеринке
Множество — как список приглашенных гостей. Имя "Иван" может быть в списке максимум один раз. Если ты добавишь его дважды, это всё равно один гость.

```python
guests = {"Иван", "Мария", "Иван", "Петр"}
print(len(guests))  # 3, а не 4
```

#### 🎨 Аналогия 3: Палитра цветов
Множество — как палитра художника. У тебя есть набор уникальных цветов: {красный, синий, желтый}. Тебе не нужно два одинаковых красных. Порядок цветов на палитре не важен, важно только их наличие.

## Основные операции с множествами

```python
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

# Объединение (все элементы из обоих)
print(a | b)  # {1, 2, 3, 4, 5, 6}
print(a.union(b))

# Пересечение (только общие)
print(a & b)  # {3, 4}
print(a.intersection(b))

# Разность (только в a, но не в b)
print(a - b)  # {1, 2}
print(a.difference(b))

# Симметрическая разность (все, кроме общих)
print(a ^ b)  # {1, 2, 5, 6}

# Добавление и удаление
a.add(10)       # Добавить элемент
a.remove(1)     # Удалить (ошибка, если нет)
a.discard(99)   # Удалить (без ошибки, если нет)
```

## Зачем это нужно

**Кортежи** используются там, где данные не должны меняться: координаты, настройки, возвращаемые значения функций. Они работают быстрее списков и защищены от случайных изменений.

**Множества** незаменимы для:
- Удаления дубликатов из списка: `list(set(my_list))`
- Быстрой проверки наличия элемента: `if x in my_set:` (работает за O(1), в отличие от списка)
- Математических операций над группами данных (общие друзья, уникальные теги).

## Типичные ошибки

### Ошибка 1: Изменение кортежа
```python
# ❌ Попытка изменить кортеж
point = (10, 20)
point[0] = 15  # TypeError: 'tuple' object does not support item assignment

# ✅ Создание нового кортежа
point = (15, 20)
```

### Ошибка 2: Обращение по индексу к множеству
```python
# ❌ У множеств нет индексов
my_set = {1, 2, 3}
print(my_set[0])  # TypeError: 'set' object is not subscriptable

# ✅ Преобразование в список, если нужен индекс
print(list(my_set)[0])
```

### Ошибка 3: Создание множества из одного элемента
```python
# ❌ Это не множество, а просто число в скобках
not_a_set = (42)  # Это просто int

# ✅ Правильный кортеж из одного элемента
single_tuple = (42,)  # Запятая обязательна!

# ✅ Правильное множество из одного элемента
single_set = {42}  # Работает
```

## Итого

**Кортеж (tuple)** — неизменяемый список. Используется для защиты данных от изменений. Синтаксис: `(1, 2, 3)`.

**Множество (set)** — коллекция уникальных элементов без индексов. Используется для удаления дубликатов и быстрых проверок. Синтаксис: `{1, 2, 3}`.

Операции множеств: объединение `|`, пересечение `&`, разность `-`.
Кортежи нельзя менять, множества можно, но только добавляя/удаляя элементы целиком.
