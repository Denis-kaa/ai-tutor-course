# Примеры кода: Переменные и типы данных

## Пример 1: Создание переменных разных типов

**Задача:** Создать переменные всех основных типов и вывести их.

```python
# Создаём переменную типа int (целое число)
age = 30                    # age теперь хранит число 30

# Создаём переменную типа float (дробное число)
height = 1.75               # height хранит число с плавающей точкой

# Создаём переменную типа str (строка)
name = "Алексей"            # name хранит текст в двойных кавычках

# Создаём переменную типа bool (логическое значение)
is_student = True           # is_student хранит True (истина)

# Создаём переменную со значением None (пустота)
middle_name = None          # middle_name хранит "ничего"

# Выводим все переменные на экран
print(name)                 # Печатает: Алексей
print(age)                  # Печатает: 30
print(height)               # Печатает: 1.75
print(is_student)           # Печатает: True
print(middle_name)          # Печатает: None
```

**Разбор:**
Python определяет тип переменной автоматически по значению. `30` без точки → `int`. `1.75` с точкой → `float`. Текст в кавычках → `str`. `True`/`False` → `bool`. `None` → специальный тип "пустота".

**Проверка типов:**
```python
# Функция type() возвращает тип переменной
print(type(age))            # <class 'int'>
print(type(height))         # <class 'float'>
print(type(name))           # <class 'str'>
print(type(is_student))     # <class 'bool'>
print(type(middle_name))    # <class 'NoneType'>
```

## Пример 2: Преобразование типов

**Задача:** Получить число от пользователя и посчитать с ним.

```python
# input() ВСЕГДА возвращает строку, даже если ввели число
age_str = input("Сколько тебе лет? ")   # Пользователь ввёл "25"

# Пытаемся сложить строку с числом — будет ошибка!
# result = age_str + 5    # TypeError: can only concatenate str to str

# Преобразуем строку в целое число через int()
age = int(age_str)        # Теперь age — это число 25, а не строка "25"

# Теперь можно считать
age_next_year = age + 1   # 25 + 1 = 26
print("В следующем году тебе будет", age_next_year)  # В следующем году тебе будет 26
```

**Разбор:**
`input()` всегда возвращает `str`. Если нужно число — преобразуй через `int()` (целое) или `float()` (дробное). Если пользователь ввёл не число, `int()` выбросит `ValueError`.

**Безопасный вариант:**
```python
# Преобразуем сразу при вводе
age = int(input("Сколько тебе лет? "))  # Короче, но упадёт при некорректном вводе

# Безопасный вариант с обработкой ошибки
try:
    age = int(input("Сколько тебе лет? "))
    print("Тебе", age, "лет")
except ValueError:
    print("Это не число!")
```

## Пример 3: Множественное присваивание

**Задача:** Создать несколько переменных одной строкой.

```python
# Способ 1: каждая переменная на своей строке
x = 10
y = 20
z = 30

# Способ 2: множественное присваивание (короче)
x, y, z = 10, 20, 30        # x=10, y=20, z=30 — всё одной строкой

# Способ 3: одинаковые значения разным переменным
a = b = c = 0               # a=0, b=0, c=0

# Обмен значений местами (без временной переменной!)
x, y = 10, 20
x, y = y, x                 # Теперь x=20, y=10
print(x, y)                 # 20 10
```

**Разбор:**
Python позволяет присваивать несколько значений за раз. Обмен `x, y = y, x` — фирменная фишка Python, в других языках для этого нужна временная переменная.

## Пример 4: f-строки для красивого вывода

**Задача:** Вывести данные о человеке в читаемом виде.

```python
# Исходные данные
name = "Мария"
age = 28
city = "Санкт-Петербург"
salary = 85000.50

# Способ 1: склеивание через + (неудобно, нужно приводить типы)
# print("Имя: " + name + ", возраст: " + str(age))  # Громоздко!

# Способ 2: f-строка (рекомендуется)
# Ставим букву f перед кавычками, переменные в фигурных скобках {}
message = f"Меня зовут {name}, мне {age} лет, я из города {city}"
print(message)  # Меня зовут Мария, мне 28 лет, я из города Санкт-Петербург

# Внутри {} можно писать выражения
print(f"Через 5 лет мне будет {age + 5}")  # Через 5 лет мне будет 33

# Форматирование чисел
print(f"Зарплата: {salary:.2f} руб.")  # Зарплата: 85000.50 руб.
print(f"Зарплата: {salary:,.0f} руб.")  # Зарплата: 85,000 руб. (с разделителями)
```

**Разбор:**
f-строки — самый современный и читаемый способ форматирования. Буква `f` перед кавычками включает режим подстановки. Всё, что в `{}`, вычисляется как выражение Python.

## Пример 5: Правила именования переменных

**Задача:** Показать правильные и неправильные имена.

```python
# ✅ ПРАВИЛЬНЫЕ имена переменных
user_name = "Иван"          # snake_case — стандарт Python
age2 = 25                   # Цифры можно, но не в начале
_private = "секрет"         # Подчёркивание в начале допустимо
totalPrice = 100            # camelCase — работает, но не по PEP 8

# ❌ НЕПРАВИЛЬНЫЕ имена (вызовут ошибку)
# 2age = 25                # SyntaxError: нельзя начинать с цифры
# user-name = "Иван"       # SyntaxError: дефис недопустим
# total price = 100        # SyntaxError: пробел недопустим
# class = "Python"         # SyntaxError: 'class' — зарезервированное слово
# def = 10                 # SyntaxError: 'def' — зарезервированное слово

# Проверка: является ли строка допустимым именем переменной
print("user_name".isidentifier())   # True
print("2age".isidentifier())        # False
print("class".isidentifier())       # True (синтаксически), но это ключевое слово!

import keyword
print(keyword.iskeyword("class"))   # True — это зарезервированное слово
```

**Разбор:**
Имя переменной может содержать буквы, цифры и `_`, но не может начинаться с цифры. Нельзя использовать зарезервированные слова Python (`if`, `for`, `class`, `def` и т.д.). Стандарт стиля — `snake_case` (слова через подчёркивание).

## Частые ошибки новичков

| Ошибка | Пример | Решение |
|--------|--------|---------|
| `NameError` | `print(usr_name)` вместо `user_name` | Проверь опечатку в имени |
| `TypeError` | `"2" + 2` | Преобразуй: `int("2") + 2` |
| `SyntaxError` | `2age = 5` | Имя не может начинаться с цифры |
| Перепутал `=` и `==` | `if x = 5:` | `=` — присваивание, `==` — сравнение |
