# Списки в Python

## Что такое список

Список (list) — это упорядоченная, изменяемая коллекция элементов. В отличие от строк, списки можно менять: добавлять, удалять или заменять элементы. Элементы могут быть любого типа, даже других списков.

```python
numbers = [1, 2, 3, 4, 5]
mixed = [1, "два", 3.0, True]
nested = [[1, 2], [3, 4]]  # Вложенный список
```

## Аналогии списка

#### 🛒 Аналогия 1: Список покупок
Самая прямая аналогия. Ты можешь добавить товар в конец (append), вставить в начало (insert), вычеркнуть (remove) или взять третий по счёту (индекс 2).

#### 🗄️ Аналогия 2: Шкаф с пронумерованными ящиками
Список — это шкаф. Ящик №0, Ящик №1 и т.д. Ты можешь положить что-то в ящик (замена по индексу), выдвинуть его (доступ), или добавить новый ящик в конец.

#### 🚂 Аналогия 3: Поезд с вагонами
Как и в строке, но с ключевым отличием: вагоны в поезде-списке можно отцеплять, прицеплять новые или менять местами прямо на ходу.

#### 📚 Аналогия 4: Книжная полка
Книги стоят в определённом порядке. Ты можешь поставить новую книгу в конец, убрать старую, или поменять две книги местами.

#### 🎒 Аналогия 5: Рюкзак путешественника
Ты кладёшь вещи по мере необходимости. Порядок важен (что лежит сверху, то достаёшь первым, если использовать как стек), и содержимое можно менять в любой момент.

## Основные операции и методы

```python
fruits = ["яблоко", "банан"]

# Добавление
fruits.append("вишня")       # ['яблоко', 'банан', 'вишня'] (в конец)
fruits.insert(0, "апельсин") # ['апельсин', 'яблоко', 'банан', 'вишня']

# Удаление
fruits.remove("банан")       # Удаляет первое вхождение
popped = fruits.pop()        # Удаляет и возвращает последний элемент ('вишня')
del fruits[0]                # Удаляет элемент по индексу

# Изменение
fruits[0] = "груша"          # Замена элемента

# Сортировка
numbers = [3, 1, 4, 1, 5]
numbers.sort()               # [1, 1, 3, 4, 5] (на месте)
numbers.reverse()            # Переворот списка
```

## List Comprehension (Генераторы списков)

Мощный и лаконичный способ создания списков.

```python
# Старый способ: создать список квадратов
squares = []
for x in range(5):
    squares.append(x**2)

# Новый способ (List Comprehension):
squares = [x**2 for x in range(5)]  # [0, 1, 4, 9, 16]

# С условием: только чётные квадраты
even_squares = [x**2 for x in range(10) if x % 2 == 0]
```

## Зачем это нужно

Списки — это основа хранения данных в Python. Без них невозможно обрабатывать наборы значений, результаты циклов или данные из файлов. List comprehension делает код короче и быстрее, чем обычные циклы `for`.

## Итого

Список — изменяемая упорядоченная коллекция.
Индексация как у строк (с 0, поддерживаются отрицательные индексы и срезы).
Основные методы: `append`, `insert`, `remove`, `pop`, `sort`.
List comprehension — элегантный способ создания списков в одну строку.
