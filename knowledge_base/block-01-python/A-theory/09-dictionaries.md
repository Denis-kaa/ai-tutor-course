# Словари в Python

## Что такое словарь

Словарь (dict) — это коллекция пар "ключ-значение". В отличие от списков, где доступ идет по индексу (0, 1, 2...), в словаре доступ идет по уникальному ключу.

```python
person = {
    "name": "Алексей",
    "age": 30,
    "city": "Москва"
}
print(person["name"])  # Алексей
```

## Аналогии словаря

#### 📖 Аналогия 1: Телефонная книга
Словарь — это как контакты в телефоне. У каждого контакта есть имя (ключ) и номер телефона (значение). Ты не ищешь "третий контакт в списке", ты ищешь "Маму" и сразу получаешь её номер.

```python
contacts = {"мама": "+79991112233", "друг": "+79994445566"}
print(contacts["мама"])  # +79991112233
```

#### 🗄️ Аналогия 2: Картотека в библиотеке
Словарь — как библиотечная картотека. На каждой карточке написано название книги (ключ) и её шифр хранения (значение). Ты называешь название — тебе сразу говорят, где искать.

#### 🏷️ Аналогия 3: Ценники в магазине
Словарь — как ценники на полках. Товар (ключ) привязан к цене (значение). Ты берешь "молоко" и сразу видишь "80 рублей".

```python
prices = {"молоко": 80, "хлеб": 45, "яйца": 120}
print(prices["хлеб"])  # 45
```

#### 🎫 Аналогия 4: Билет в кино
Словарь — как билет в кинотеатр. У тебя есть номер ряда (ключ) и номер места (значение). Или наоборот: место "А5" (ключ) -> фильм "Матрица" (значение).

#### 🧩 Аналогия 5: Пазл с подписями
Словарь — как набор деталей пазла, где на каждой детали написан её номер (ключ), а внутри — картинка (значение). Ты не перебираешь все детали подряд, а сразу берешь нужную по номеру.

## Основные операции и методы

```python
user = {"name": "Иван", "age": 25}

# Доступ к значению
print(user["name"])       # Иван
print(user.get("age"))    # 25
print(user.get("email", "не указан"))  # "не указан" (безопасный доступ)

# Добавление / Изменение
user["email"] = "ivan@mail.ru"  # Добавление нового ключа
user["age"] = 26                # Изменение существующего

# Удаление
del user["email"]               # Удаляет ключ и значение
age = user.pop("age")           # Удаляет и возвращает значение

# Перебор словаря
for key, value in user.items():
    print(f"{key}: {value}")

# Только ключи или только значения
print(user.keys())    # dict_keys(['name'])
print(user.values())  # dict_values(['Иван'])
```

## Dict Comprehension (Генераторы словарей)

Как и со списками, словари можно создавать в одну строку:

```python
# Старый способ
squares = {}
for x in range(5):
    squares[x] = x**2

# Новый способ
squares = {x: x**2 for x in range(5)}  # {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}
```

## Зачем это нужно

Словари — это основа работы с данными в Python. JSON-файлы, ответы API, настройки программ, базы данных — всё это преобразуется в словари. Без них пришлось бы использовать два параллельных списка и постоянно синхронизировать их индексы, что крайне неудобно.

## Типичные ошибки

### Ошибка 1: KeyError
```python
# ❌ Обращение к несуществующему ключу
data = {"a": 1}
print(data["b"])  # KeyError: 'b'

# ✅ Безопасный доступ через .get()
print(data.get("b", "значение по умолчанию"))
```

### Ошибка 2: Изменяемые ключи
```python
# ❌ Список не может быть ключом
# bad_dict = {[1, 2]: "value"}  # TypeError

# ✅ Кортеж может быть ключом
good_dict = {(1, 2): "value"}  # Работает!
```

## Итого

Словарь — коллекция пар "ключ-значение".
Ключи должны быть уникальными и неизменяемыми (строки, числа, кортежи).
Доступ по ключу: `dict[key]` или безопасно `dict.get(key)`.
Методы: `keys()`, `values()`, `items()`, `pop()`, `update()`.
Dict comprehension — быстрый способ создания словарей.
