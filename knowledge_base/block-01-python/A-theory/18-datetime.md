# Работа с датами и временем в Python

## Что такое модуль datetime

Модуль `datetime` — встроенный инструмент для работы с датами и временем. Он позволяет получать текущее время, создавать даты, вычислять разницу между ними и форматировать вывод.

Основные классы:
- `date` — только дата (год, месяц, день)
- `time` — только время (часы, минуты, секунды)
- `datetime` — дата и время вместе
- `timedelta` — промежуток времени (разница между датами)

```python
from datetime import datetime, date, timedelta

now = datetime.now()
print(now)  # 2026-07-24 14:30:15.123456

today = date.today()
print(today)  # 2026-07-24
```

## Аналогии работы с датами

#### 📅 Аналогия 1: Настенный календарь
`date` — как листок настенного календаря. На нём написано число, месяц, год. Ты можешь посмотреть сегодняшнюю дату, перелистнуть на завтра (добавить день) или посчитать, сколько дней до Нового года.

```python
today = date.today()
new_year = date(2027, 1, 1)
days_left = (new_year - today).days
print(f"До Нового года {days_left} дней")
```

#### ⏰ Аналогия 2: Будильник
`time` — как будильник. Он хранит часы, минуты, секунды. Ты можешь завести его на 7:30, и он сработает в нужный момент.

```python
from datetime import time
alarm = time(7, 30, 0)  # 7 часов 30 минут 0 секунд
print(alarm)  # 07:30:00
```

#### ⏳ Аналогия 3: Песочные часы (timedelta)
`timedelta` — как песочные часы или таймер. Это не конкретный момент, а **промежуток** времени: "3 дня", "2 часа", "30 минут". Ты можешь прибавить промежуток к дате и получить новую дату.

```python
deadline = datetime.now() + timedelta(days=7)
print(f"Дедлайн через неделю: {deadline}")
```

#### 🚂 Аналогия 4: Расписание поездов
`datetime` — как расписание на вокзале. У каждого поезда есть точная дата и время отправления. Ты можешь сравнить два поезда, посчитать разницу, узнать, какой приедет раньше.

#### 📝 Аналогия 5: Дневник с датами
`datetime` — как записи в дневнике. Каждая запись привязана к конкретному моменту. Ты можешь отсортировать записи по датам, найти записи за последний месяц, посчитать, сколько дней прошло между событиями.

## Форматирование дат (strftime)

`strftime` превращает объект даты в красивую строку по шаблону.

```python
now = datetime.now()

print(now.strftime("%d.%m.%Y"))        # 24.07.2026
print(now.strftime("%H:%M:%S"))        # 14:30:15
print(now.strftime("%A, %d %B %Y"))    # Thursday, 24 July 2026
print(now.strftime("%Y-%m-%d %H:%M"))  # 2026-07-24 14:30
```

Основные коды формата:
| Код | Значение | Пример |
|-----|----------|--------|
| `%Y` | Год (4 цифры) | 2026 |
| `%m` | Месяц (01-12) | 07 |
| `%d` | День (01-31) | 24 |
| `%H` | Часы (00-23) | 14 |
| `%M` | Минуты (00-59) | 30 |
| `%S` | Секунды (00-59) | 15 |
| `%A` | День недели | Thursday |
| `%B` | Название месяца | July |

## Парсинг дат из строк (strptime)

`strptime` — обратная операция: превращает строку в объект даты.

```python
date_string = "24.07.2026"
parsed = datetime.strptime(date_string, "%d.%m.%Y")
print(parsed.year)   # 2026
print(parsed.month)  # 7
```

## Зачем это нужно

Без работы с датами невозможно:
- Логи с временными метками
- Напоминания и планировщики
- Подсчёт возраста, стажа, сроков
- Анализ данных по периодам
- Работа с API (даты в JSON)

## Типичные ошибки

### Ошибка 1: datetime vs date
```python
from datetime import datetime, date

# ❌ Путаница классов
d = date.today()
# d.hour  # AttributeError! У date нет часов

# ✅ Используй datetime для времени
dt = datetime.now()
print(dt.hour)  # Работает
```

### Ошибка 2: Неправильный формат в strptime
```python
# ❌ Формат не совпадает со строкой
datetime.strptime("24/07/2026", "%d.%m.%Y")  # ValueError!

# ✅ Формат должен точно совпадать
datetime.strptime("24/07/2026", "%d/%m/%Y")  # Работает
```

### Ошибка 3: Вычитание дат без timedelta
```python
# ❌ Нельзя просто вычесть числа из даты
# date.today() - 7  # TypeError!

# ✅ Используй timedelta
from datetime import timedelta
week_ago = date.today() - timedelta(days=7)
```

## Итого

`datetime.now()` — текущие дата и время.
`date.today()` — только сегодняшняя дата.
`timedelta(days=7)` — промежуток времени для арифметики дат.
`strftime("%d.%m.%Y")` — дата → строка.
`strptime("24.07.2026", "%d.%m.%Y")` — строка → дата.
Разница дат: `(date2 - date1).days` — количество дней.
