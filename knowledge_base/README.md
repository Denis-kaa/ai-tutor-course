# База знаний AI-Тьютора

Полная база знаний для 17 блоков курса "Практический курс разработчика".
Каждый блок содержит 9 разделов (A-I) общим объёмом 100 000+ слов.

## Структура блока

```
block-XX-name/
├── A-theory/           # Теория с аналогиями (20 000 слов)
├── B-code-examples/    # Примеры кода с комментариями (20 000 слов)
├── C-dialogues/        # Диалоги "Ментор — Студент" (20 000 слов)
├── D-errors/           # Разборы типичных ошибок (15 000 слов)
├── E-exercises/        # Упражнения с решениями (10 000 слов)
├── F-faq/              # Часто задаваемые вопросы (5 000 слов)
├── G-glossary/         # Словарь терминов (3 000 слов)
├── H-cheatsheets/      # Шпаргалки и чек-листы (3 000 слов)
└── I-cases/            # Реальные кейсы и истории (4 000 слов)
```
