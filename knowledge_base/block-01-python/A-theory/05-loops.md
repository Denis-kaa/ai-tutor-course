# Циклы в Python

## Что такое циклы

Цикл — это конструкция, которая позволяет выполнять один и тот же блок кода многократно. Без циклов тебе пришлось бы копировать и вставлять один и тот же код десятки раз.

Два основных типа циклов в Python:
- **for** — когда заранее знаешь, сколько раз повторять (или есть коллекция элементов)
- **while** — когда нужно повторять, пока условие истинно (не знаешь заранее, сколько раз)

## Цикл for

### Базовый синтаксис

```python
# Перебор элементов списка
fruits = ["яблоко", "банан", "вишня"]
for fruit in fruits:
    print(fruit)
# яблоко
# банан
# вишня

# Перебор чисел через range()
for i in range(5):
    print(i)
# 0, 1, 2, 3, 4
```

### Аналогии цикла for

#### 🏭 Аналогия 1: Конвейер на заводе

Цикл for — это конвейер. На ленте лежат детали (элементы списка). Конвейер берёт каждую деталь по очереди и выполняет с ней одну и ту же операцию. Когда лента пуста — конвейер останавливается.

```python
products = ["телефон", "ноутбук", "наушники"]
for product in products:
    print(f"Упаковываем: {product}")
```

#### 📬 Аналогия 2: Почтальон с сумкой писем

Цикл for — как почтальон, у которого в сумке 10 писем. Он берёт первое письмо, идёт по адресу, опускает в ящик. Берёт второе — и так далее, пока сумка не опустеет.

```python
addresses = ["ул. Ленина 5", "ул. Мира 12", "пр. Победы 3"]
for addr in addresses:
    print(f"Доставляю письмо по адресу: {addr}")
```

#### 🍽️ Аналогия 3: Мытьё посуды

Цикл for — как мытьё посуды после ужина. Ты берёшь каждую тарелку по одной, моешь, ставишь на сушку. Когда все тарелки вымыты — заканчиваешь.

#### 🎵 Аналогия 4: Плейлист в наушниках

Цикл for — как воспроизведение плейлиста. Музыкальный проигрыватель берёт первый трек, играет до конца, берёт второй, и так до последнего трека.

#### 🛒 Аналогия 5: Кассир в супермаркете

Цикл for — как кассир, пробивающий товары. Берёт первый товар — пикает штрихкод. Берёт второй — пикает. Когда все товары пробиты — говорит общую сумму.

## Цикл while

### Базовый синтаксис

```python
count = 0
while count < 5:
    print(f"Счётчик: {count}")
    count += 1  # Обязательно! Иначе бесконечный цикл
```

### Аналогии цикла while

#### 🔁 Аналогия 1: Стиральная машина

Цикл while — как стиральная машина. Она крутит барабан, **пока** таймер не покажет 0. Ты не знаешь заранее, сколько оборотов будет — машина сама остановится, когда условие "время > 0" станет ложным.

#### 🎮 Аналогия 2: Жизни в игре

Цикл while — как жизни в видеоигре. Ты играешь, **пока** у тебя есть жизни. Потерял последнюю — игра окончена.

#### 🚗 Аналогия 3: Езда на бензине

Цикл while — как езда на машине. Ты едешь, **пока** в баке есть бензин. Кончился бензин — машина встала.

## Управление циклами: break и continue

### break — досрочный выход

```python
# Ищем первый чётный элемент
numbers = [1, 3, 7, 4, 9, 2]
for num in numbers:
    if num % 2 == 0:
        print(f"Первый чётный: {num}")
        break  # Нашли — выходим из цикла
```

**Аналогия:** break — как кнопка "Стоп" на беговой дорожке. Нажал — и сразу остановился.

### continue — пропуск итерации

```python
# Печатаем только нечётные
for i in range(10):
    if i % 2 == 0:
        continue  # Пропускаем чётные
    print(i)  # 1, 3, 5, 7, 9
```

**Аналогия:** continue — как когда ты перебираешь яблоки в корзине. Видишь гнилое — пропускаешь, берёшь следующее.

## Типичные ошибки

### Ошибка 1: Бесконечный while

```python
# ❌ Бесконечный цикл!
count = 0
while count < 5:
    print(count)
    # Забыл count += 1!
```

### Ошибка 2: Изменение списка в цикле for

```python
# ❌ Опасно! Список меняется во время перебора!
items = [1, 2, 3, 4, 5]
for item in items:
    if item % 2 == 0:
        items.remove(item)

# ✅ Правильно: создать новый список
items = [x for x in items if x % 2 != 0]
```

## Итого

**for** — для перебора элементов коллекции или диапазона чисел. Знаешь, сколько итераций — используй for.
**while** — для повторения, пока условие истинно. Не знаешь, сколько итераций — используй while.
**break** — досрочный выход из цикла.
**continue** — пропуск текущей итерации.
Главное правило: всегда проверяй, что цикл когда-нибудь закончится!
