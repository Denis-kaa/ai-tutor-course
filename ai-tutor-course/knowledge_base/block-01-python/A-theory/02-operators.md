# Операторы в Python

## Что такое операторы

Оператор — это символ или комбинация символов, которая выполняет операцию над значениями (операндами). Если переменная — это коробка с данными, то оператор — это действие, которое мы с этими данными производим: сложить, сравнить, проверить условие.

В Python есть три основные группы операторов:
- **Арифметические** — для математических вычислений (+, -, *, /)
- **Операторы сравнения** — для сравнения значений (==, >, <)
- **Логические** — для комбинации условий (and, or, not)

## Арифметические операторы

### Основные арифметические операторы

\`\`\`python
a = 10
b = 3

print(a + b)   # 13 — сложение
print(a - b)   # 7  — вычитание
print(a * b)   # 30 — умножение
print(a / b)   # 3.333... — деление (всегда float!)
print(a // b)  # 3  — целочисленное деление (отбрасывает дробную часть)
print(a % b)   # 1  — остаток от деления (модуль)
print(a ** b)  # 1000 — возведение в степень (10 в степени 3)
\`\`\`

### Аналогии арифметических операторов

#### 🧮 Аналогия 1: Калькулятор на телефоне

Арифметические операторы — это кнопки калькулятора. \`+\` — кнопка сложения, \`-\` — вычитания, \`*\` — умножения. Ты нажимаешь кнопки, вводишь числа, получаешь результат.

\`\`\`python
price = 100
quantity = 5
total = price * quantity  # Нажал "×" — получил 500
\`\`\`

**Что это даёт:** Не нужно считать в уме. Компьютер делает это мгновенно и без ошибок.

#### 🍕 Аналогия 2: Деление пиццы

Оператор \`/\` — это когда ты делишь пиццу на равные части. 10 кусков на 3 человек = каждому по 3.33 куска (float).

Оператор \`//\` — это когда ты раздаёшь целые куски, а дробные отбрасываешь. 10 // 3 = 3 куска каждому (остаток не важен).

Оператор \`%\` — это сколько кусков осталось после того, как всем раздал поровну. 10 % 3 = 1 кусок в остатке.

\`\`\`python
slices = 10
people = 3

each_gets = slices // people  # 3 целых куска
remainder = slices % people   # 1 кусок в остатке

print(f"Каждому по {each_gets} куска, осталось {remainder}")
\`\`\`

#### 💰 Аналогия 3: Сдача в магазине

Оператор \`%\` (остаток от деления) — это как сдача. Ты дал 100 рублей, товар стоит 35 рублей. Сколько сдачи? \`100 % 35 = 30\`.

\`\`\`python
paid = 100
price = 35
change = paid % price  # 30 рублей сдачи
\`\`\`

**Практическое применение:** Оператор \`%\` часто используется для проверки чётности:
\`\`\`python
number = 7
if number % 2 == 0:
    print("Чётное")
else:
    print("Нечётное")  # Выведет это, так как 7 % 2 = 1
\`\`\`

#### 📈 Аналогия 4: Рост вклада в банке

Оператор \`**\` (возведение в степень) — это как сложный процент. Вложил 1000 рублей под 10% годовых на 3 года:

\`\`\`python
principal = 1000
rate = 1.10  # 10% годовых
years = 3

final = principal * (rate ** years)  # 1000 * 1.331 = 1331
print(f"Через {years} года будет {final} рублей")
\`\`\`

#### 🧩 Аналогия 5: Пазл из действий

Арифметические операторы — как детали конструктора. Из них ты собираешь сложные вычисления, как из деталей Lego — замок или машину.

\`\`\`python
# Сложная формула: площадь треугольника
base = 10
height = 5
area = (base * height) / 2  # 25.0

# Скобки меняют порядок — как разные способы соединения деталей
result1 = 2 + 3 * 4    # 14 (сначала умножение)
result2 = (2 + 3) * 4  # 20 (сначала сложение в скобках)
\`\`\`

## Операторы сравнения

Операторы сравнения возвращают \`True\` или \`False\`:

\`\`\`python
x = 10
y = 5

print(x == y)  # False — равно
print(x != y)  # True  — не равно
print(x > y)   # True  — больше
print(x < y)   # False — меньше
print(x >= y)  # True  — больше или равно
print(x <= y)  # False — меньше или равно
\`\`\`

### Аналогии операторов сравнения

#### ⚖️ Аналогия 1: Весы в магазине

Операторы сравнения — это весы. Ты кладёшь на одну чашу одно значение, на другую — другое. Весы показывают: перевешивает ли одна чаша (>, <), или они в равновесии (==).

\`\`\`python
weight_apple = 200  # грамм
weight_orange = 150

if weight_apple > weight_orange:
    print("Яблоко тяжелее")  # Выведет это
\`\`\`

#### 🎓 Аналогия 2: Проходной балл на экзамене

Оператор \`>=\` — это как проходной балл. Чтобы поступить, нужно набрать **не менее** 200 баллов.

\`\`\`python
score = 215
passing_score = 200

if score >= passing_score:
    print("Поздравляю, ты поступил!")
else:
    print("В этот раз не получилось")
\`\`\`

#### 🔍 Аналогия 3: Поиск отличий на картинках

Оператор \`==\` — это как игра "найди 10 отличий". Ты сравниваешь две картинки (значения) и проверяешь, **идентичны** ли они.

\`\`\`python
password_input = "secret123"
password_stored = "secret123"

if password_input == password_stored:
    print("Пароль верный, вход разрешён")
\`\`\`

**Важно:** \`==\` — это сравнение (равно ли?), а \`=\` — это присваивание (положи значение в переменную). Путать их — самая частая ошибка новичков!

#### 🏁 Аналогия 4: Финишная черта на соревнованиях

Оператор \`<\` — это как проверка, добежал ли спортсмен до финиша. Если его время **меньше** рекорда — он победил.

\`\`\`python
runner_time = 9.58  # секунды
record = 9.69

if runner_time < record:
    print("Новый мировой рекорд!")
\`\`\`

#### 🎫 Аналогия 5: Контролёр в метро

Оператор \`!=\` (не равно) — это как контролёр, который проверяет: "Твой билет **не равен** просроченному? Значит, проходи".

\`\`\`python
ticket_status = "valid"

if ticket_status != "expired":
    print("Проходите")
\`\`\`

## Логические операторы

Логические операторы объединяют несколько условий:

\`\`\`python
age = 25
has_license = True
has_car = False

# and — оба условия должны быть True
can_drive = age >= 18 and has_license  # True

# or — хотя бы одно условие должно быть True
has_transport = has_car or has_license  # True

# not — инвертирует значение
is_minor = not (age >= 18)  # False
\`\`\`

### Аналогии логических операторов

#### 🔌 Аналогия 1: Электрическая цепь

Оператор \`and\` — это как два выключателя, соединённых последовательно. Лампочка загорится **только если оба** выключателя включены.

\`\`\`python
switch_1 = True
switch_2 = True
light = switch_1 and switch_2  # True — лампочка горит

switch_1 = False
light = switch_1 and switch_2  # False — цепь разорвана
\`\`\`

#### 🎁 Аналогия 2: Подарок на день рождения

Оператор \`or\` — это как выбор подарка. Ты получишь подарок, **если** друг придёт **или** пришлёт его курьером. Достаточно одного условия.

\`\`\`python
friend_came = False
courier_arrived = True
got_gift = friend_came or courier_arrived  # True — подарок получен
\`\`\`

#### 🚫 Аналогия 3: Запрещающий знак

Оператор \`not\` — это как запрещающий знак. "Не входить" = инверсия "входить".

\`\`\`python
is_raining = True
go_for_walk = not is_raining  # False — не идём гулять
\`\`\`

#### 🎯 Аналогия 4: Приём на работу

Комбинация операторов — как требования к кандидату:

\`\`\`python
age = 30
experience = 5
education = "высшее"

# Должно быть: возраст 25-40 И опыт >= 3 И высшее образование
is_suitable = (25 <= age <= 40) and (experience >= 3) and (education == "высшее")

if is_suitable:
    print("Кандидат подходит!")
\`\`\`

#### 🚦 Аналогия 5: Светофор с дополнительными условиями

Представь перекрёсток: можно ехать, если горит зелёный **И** нет пешеходов **ИЛИ** ты спецтранспорт.

\`\`\`python
light = "green"
has_pedestrians = False
is_emergency = False

can_go = (light == "green" and not has_pedestrians) or is_emergency
\`\`\`

## Приоритет операторов

Как в математике: сначала скобки, потом степени, потом умножение/деление, потом сложение/вычитание.

\`\`\`python
# Без скобок — приоритет у умножения
result = 2 + 3 * 4  # 14, а не 20

# Со скобками — меняем порядок
result = (2 + 3) * 4  # 20

# Логические операторы: not > and > or
result = True or False and False  # True (сначала and)
result = (True or False) and False  # False (скобки изменили порядок)
\`\`\`

## Зачем это нужно

Без операторов мы бы не могли:
- Считать зарплаты, налоги, скидки
- Сравнивать значения (больше/меньше/равно)
- Принимать решения в программе (если то-то, то сделать то-то)
- Строить сложную логику (если А и Б, но не В)

## Что было бы, если бы операторов не существовало

Представь, что ты пишешь программу для расчёта скидки в магазине. Без операторов:

\`\`\`python
# Без операторов — приходится считать вручную
price = 1000
# "Так, скидка 20%... это 1000 делить на 5... будет 200... отнять от 1000... 800"
final_price = 800  # Вбил руками — ужас!
\`\`\`

**С операторами:**
\`\`\`python
price = 1000
discount_percent = 20
discount = price * discount_percent / 100  # 200
final_price = price - discount  # 800
\`\`\`

Изменил процент — и всё пересчиталось автоматически.

## Связь с другими концепциями

Операторы тесно связаны с:
- **Переменными** — операторы работают со значениями в переменных
- **Условиями** (if) — используют операторы сравнения и логические
- **Циклами** — используют арифметические операторы для счётчиков
- **Функциями** — возвращают результат вычислений

Операторы — это "глаголы" языка программирования. Если переменные — это "существительные" (данные), то операторы — это действия над ними.

## Итого

Операторы — это символы для выполнения операций над данными.

**Арифметические:** \`+\`, \`-\`, \`*\`, \`/\`, \`//\`, \`%\`, \`**\`
**Сравнения:** \`==\`, \`!=\`, \`>\`, \`<\`, \`>=\`, \`<=\`
**Логические:** \`and\`, \`or\`, \`not\`

Важно:
- \`==\` — сравнение, \`=\` — присваивание (не путать!)
- \`//\` — целочисленное деление, \`/\` — обычное (всегда float)
- \`%\` — остаток от деления (полезно для проверки чётности)
- Приоритет операторов можно менять скобками
