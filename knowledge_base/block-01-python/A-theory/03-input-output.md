# Ввод и вывод данных в Python

## Что такое ввод/вывод (I/O)

Ввод (Input) — это получение данных от пользователя или из внешнего источника. Вывод (Output) — это отображение результатов работы программы.

Без ввода/вывода программа была бы "чёрным ящиком": что-то считает внутри, но ты не можешь ни дать ей данные, ни увидеть результат.

Основные функции:
- \`print()\` — вывод данных на экран
- \`input()\` — ввод данных от пользователя

## Функция print()

### Базовый вывод

\`\`\`python
print("Привет, мир!")        # Строка
print(42)                    # Число
print(3.14)                  # Дробное число
print(True)                  # Логическое значение
print("Возраст:", 25)        # Несколько значений через запятую
\`\`\`

### Аналогии функции print()

#### 📢 Аналогия 1: Громкоговоритель на стадионе

Функция \`print()\` — это громкоговоритель. Ты говоришь в микрофон (передаёшь данные), и все на стадионе (на экране) слышат твоё сообщение.

\`\`\`python
message = "Внимание, старт!"
print(message)  # Громкоговоритель воспроизвёл сообщение
\`\`\`

**Что это даёт:** Ты можешь "сообщить" пользователю о результатах работы программы, ошибках, статусах.

#### 📝 Аналогия 2: Печать документа на принтере

\`print()\` — это как отправить документ на принтер. Ты создаёшь текст в компьютере, нажимаешь "Печать" — и получаешь физическую копию.

\`\`\`python
report = "Отчёт за май: прибыль 100 000 руб."
print(report)  # "Распечатал" отчёт на экран
\`\`\`

#### 🗣️ Аналогия 3: Речь учителя в классе

Учитель (программа) объясняет материал ученикам (пользователю). \`print()\` — это его голос.

\`\`\`python
print("Сегодня мы изучаем переменные.")
print("Переменная — это коробка с именем.")
print("Давайте разберём примеры.")
\`\`\`

#### 📺 Аналогия 4: Субтитры в фильме

\`print()\` — это субтитры, которые появляются на экране и объясняют, что происходит.

\`\`\`python
print("[Сцена 1: Герой входит в комнату]")
print("Герой: Привет!")
print("[Сцена 2: Герой садится за стол]")
\`\`\`

#### 📣 Аналогия 5: Объявление по радио

\`print()\` — как объявление по радио. Короткое, чёткое сообщение для всех слушателей.

\`\`\`python
print("Внимание! Поезд прибывает на первый путь.")
\`\`\`

### Параметры print()

\`\`\`python
# sep — разделитель между значениями (по умолчанию пробел)
print("2026", "07", "24", sep="-")  # 2026-07-24
print("Иван", "Иванов", sep=". ")   # Иван. Иванов

# end — что добавить в конце (по умолчанию перенос строки \n)
print("Привет", end=" ")
print("мир!")  # Привет мир! (в одну строку)

# Вывод в файл
with open("log.txt", "w") as f:
    print("Запись в лог", file=f)
\`\`\`

## Функция input()

### Базовый ввод

\`\`\`python
name = input()  # Ждёт, пока пользователь введёт текст и нажмёт Enter
print("Привет,", name)
\`\`\`

**Важно:** \`input()\` всегда возвращает **строку** (str), даже если пользователь ввёл число!

\`\`\`python
age = input("Сколько тебе лет? ")  # Пользователь ввёл 25
print(type(age))  # <class 'str'>, а не int!

# Нужно преобразовать в число:
age = int(input("Сколько тебе лет? "))  # Теперь age — это int
print("Через 5 лет тебе будет", age + 5)
\`\`\`

### Аналогии функции input()

#### 🎤 Аналогия 1: Микрофон для вопросов

\`input()\` — это как микрофон, который ты протягиваешь пользователю с вопросом. Пользователь отвечает в микрофон, и ты получаешь его ответ.

\`\`\`python
answer = input("Как тебя зовут? ")  # Протянул микрофон с вопросом
print(f"Приятно познакомиться, {answer}!")
\`\`\`

#### 📝 Аналогия 2: Анкета для заполнения

\`input()\` — это строка в анкете. Ты задаёшь вопрос (подсказка), пользователь вписывает ответ.

\`\`\`python
print("=== АНКЕТА ===")
name = input("Имя: ")
age = input("Возраст: ")
city = input("Город: ")

print(f"\nСпасибо, {name} из города {city}!")
\`\`\`

#### 🎰 Аналогия 3: Приём монет в автомате

\`input()\` — как щель для приёма монет в торговом автомате. Ты ждёшь, пока пользователь "бросит монету" (введёт данные), и только потом продолжаешь работу.

\`\`\`python
print("Введите пароль:")
password = input()  # Программа ждёт, пока пользователь введёт пароль
if password == "secret":
    print("Доступ разрешён")
\`\`\`

#### 📞 Аналогия 4: Ожидание на телефонной линии

\`input()\` — как когда ты звонишь и слышишь гудки. Программа "ждёт на линии", пока пользователь не "снимет трубку" (не введёт данные).

\`\`\`python
print("Ожидание ввода...")
data = input()  # Программа останавливается и ждёт
print("Получено:", data)
\`\`\`

#### 🎯 Аналогия 5: Интерактивная игра "Вопрос-ответ"

\`input()\` — как ход в игре. Ты задал вопрос — ждёшь ответа пользователя. Без ответа игра не продолжается.

\`\`\`python
print("Загадка: Зимой и летом одним цветом. Что это?")
answer = input("Твой ответ: ")
if answer.lower() == "ёлка":
    print("Правильно!")
else:
    print("Не угадал!")
\`\`\`

## Форматирование строк

### Способ 1: f-строки (рекомендуется)

Самый современный и читаемый способ:

\`\`\`python
name = "Алексей"
age = 30

# f-строка: f"текст {переменная}"
print(f"Меня зовут {name}, мне {age} лет")

# Можно использовать выражения
print(f"Через 5 лет мне будет {age + 5}")
print(f"Имя в верхнем регистре: {name.upper()}")

# Форматирование чисел
price = 1234.5678
print(f"Цена: {price:.2f} руб.")  # 1234.57 (2 знака после запятой)
print(f"Число с разделителями: {1234567:,}")  # 1,234,567
print(f"Процент: {0.75:.1%}")  # 75.0%
\`\`\`

### Способ 2: метод format()

\`\`\`python
name = "Алексей"
age = 30

print("Меня зовут {}, мне {} лет".format(name, age))
print("Меня зовут {0}, мне {1} лет".format(name, age))  # По позиции
print("Меня зовут {name}, мне {age} лет".format(name=name, age=age))  # По имени
\`\`\`

### Способ 3: оператор % (устаревший, но встречается)

\`\`\`python
name = "Алексей"
age = 30

print("Меня зовут %s, мне %d лет" % (name, age))
# %s — строка, %d — целое число, %f — дробное
\`\`\`

### Аналогии форматирования строк

#### 📋 Аналогия 1: Шаблон документа

f-строка — это как шаблон договора с пропусками. В шаблоне написано: "Я, \`{имя}\`, паспорт \`{номер}\`...", и ты вписываешь конкретные данные.

\`\`\`python
template = f"Я, {name}, паспорт {passport_number}, проживаю в {city}"
print(template)
\`\`\`

#### 🎨 Аналогия 2: Раскраска с номерами

\`format()\` — как раскраска, где каждый элемент имеет свой номер. "В ячейку 1 вставь имя, в ячейку 2 — возраст".

\`\`\`python
print("Имя: {0}, Возраст: {1}, Город: {2}".format("Алексей", 30, "Москва"))
\`\`\`

#### 🧩 Аналогия 3: Пазл из частей

Форматирование — это как собрать пазл. У тебя есть части (переменные), и ты складываешь их в единую картинку (строку).

\`\`\`python
greeting = f"Привет, {name}! Тебе {age} лет. Ты из {city}."
\`\`\`

#### 📮 Аналогия 4: Посылка с вложениями

\`format()\` — как посылка, куда ты кладёшь вложения в определённом порядке. "В конверт 1 — фото, в конверт 2 — документы".

\`\`\`python
message = "Уважаемый {0}! Ваш заказ №{1} отправлен.".format("Иван", 12345)
\`\`\`

#### 🏷️ Аналогия 5: Бирки на товарах

f-строка — как бирка на товаре, где в пропуски вписаны конкретные значения: "Цена: \`{price}\` руб., Страна: \`{country}\`".

\`\`\`python
label = f"Товар: {product_name}, Цена: {price:.2f} руб."
\`\`\`

## Практические примеры

### Пример 1: Простой калькулятор

\`\`\`python
print("=== КАЛЬКУЛЯТОР ===")
a = float(input("Введите первое число: "))
b = float(input("Введите второе число: "))

print(f"\n{a} + {b} = {a + b}")
print(f"{a} - {b} = {a - b}")
print(f"{a} * {b} = {a * b}")
print(f"{a} / {b} = {a / b:.2f}")
\`\`\`

### Пример 2: Анкета с форматированием

\`\`\`python
print("=== ЗАПОЛНИ АНКЕТУ ===")
name = input("Имя: ")
age = int(input("Возраст: "))
city = input("Город: ")

print("\n" + "="*30)
print(f"Имя:    {name}")
print(f"Возраст: {age}")
print(f"Город:  {city}")
print(f"Год рождения: {2026 - age}")
print("="*30)
\`\`\`

### Пример 3: Форматирование таблицы

\`\`\`python
products = [
    ("Молоко", 80.50),
    ("Хлеб", 45.00),
    ("Яйца", 120.99),
]

print(f"{\"Товар\":<15} {\"Цена\":>10}")
print("-" * 27)
for name, price in products:
    print(f"{name:<15} {price:>10.2f}")
\`\`\`

## Зачем это нужно

Без ввода/вывода:
- Программа не могла бы общаться с пользователем
- Ты не мог бы создавать интерактивные приложения
- Отладка была бы невозможна (не видишь, что происходит внутри)
- Результаты вычислений оставались бы "внутри" программы

## Что было бы, если бы ввода/вывода не существовало

Представь, что ты написал программу для расчёта зарплаты. Она считает, но ты не можешь:
- Ввести данные о сотруднике (часы, ставка)
- Увидеть результат (сколько денег)

Пришлось бы каждый раз менять код вручную:
\`\`\`python
# Без input() — меняешь код каждый раз
hours = 40  # Пришлось руками вписать
rate = 500  # И здесь
salary = hours * rate
# Без print() — результат не увидишь, придётся заглядывать в отладчик
\`\`\`

С вводом/выводом — одна программа работает для любых данных.

## Связь с другими концепциями

Ввод/вывод связан с:
- **Переменными** — \`input()\` сохраняет данные в переменные
- **Типами данных** — \`input()\` возвращает строку, нужно преобразовывать
- **Условиями** — можно проверять введённые данные
- **Циклами** — можно запрашивать данные повторно, пока не будут правильными
- **Файлами** — \`print()\` может писать в файл, данные можно читать из файла

## Итого

\`print()\` — вывод данных на экран. Параметры: \`sep\` (разделитель), \`end\` (окончание), \`file\` (куда писать).

\`input()\` — ввод данных от пользователя. Всегда возвращает строку! Для чисел нужно преобразование: \`int()\` или \`float()\`.

Форматирование строк:
- **f-строки** (рекомендуется): \`f"Текст {переменная}"\`
- **format()**: \`"Текст {}".format(значение)\`
- **%** (устарел): \`"Текст %s" % значение\`
